📊 Pakistan Import Export Dashboard (Power BI)
📌 Overview

This project is an interactive Power BI dashboard that analyzes Pakistan’s import and export data.
It provides insights into trade balance, top trading partners, product categories, and yearly/monthly trends.

🎯 Objectives
Analyze import vs export performance
Identify top trading countries
Understand trade balance trends
Visualize product category distribution
🛠 Tools Used
Power BI
Power Query
DAX
Excel (Data source)

📈 Key Features
Total Imports, Exports & Trade Balance KPIs
Yearly and Monthly Trend Analysis
Top Countries by Trade Volume
Product Category Insights
Interactive filters (Year, Country, Category)

📊 Insights
Imports are generally higher than exports
A few countries dominate trade relationships
Clear seasonal patterns in trade activity

📁 Files Included
Pakistan_Import_Export.pbix → Power BI dashboard file
dataset.xlsx → Cleaned dataset
dashboard_screenshot.png → Dashboard preview

🚀 How to Use
Download the .pbix file
Open it in Power BI Desktop
Refresh data if needed
Explore using filters and visuals
